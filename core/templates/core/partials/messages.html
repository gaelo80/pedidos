{% comment %} core/partials/messages.html Muestra los mensajes de Django como toasts flotantes (esquina superior derecha). - Éxito / info: se cierran solos a los 5 s. - Error / advertencia: quedan fijos hasta que el usuario los cierra con la X. Usa el sistema de toasts de Bootstrap 5 (cargado al final de base.html). IMPORTANTE: este partial se incluye antes de que se cargue bootstrap.bundle.js, por eso el script espera a 'DOMContentLoaded' para inicializar los toasts; así 'bootstrap' ya existe cuando se ejecuta. {% endcomment %} {% if messages %}
